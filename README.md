
# ⚛️ React — Introducción al desarrollo con componentes

## ⚛️ Descripción  
Presentación sobre los fundamentos de **React**, la biblioteca de JavaScript para crear interfaces modernas, interactivas y reutilizables.  
Está pensada para quienes comienzan a aprender desarrollo web y quieren entender cómo React organiza la interfaz mediante componentes, estado y JSX.

## 📑 Diapositivas de la presentación
[Descargar presentación en PDF](./assets/REACT.pdf)

### ✔️ ¿Qué es React?
- Biblioteca para construir interfaces de usuario dinámicas.  
- Basada en componentes reutilizables.  
- Usa JSX, una mezcla de HTML y JavaScript.  
- Permite crear aplicaciones interactivas sin recargar la página.

### ✔️ Componentes  
- Un componente es una función que devuelve lo que se ve en pantalla.  
- Representan piezas independientes de la interfaz (botón, tarjeta, menú…).  
- Se pueden anidar para crear estructuras más complejas.

### ✔️ JSX  
- Sintaxis que combina HTML + JavaScript.  
- Permite escribir interfaces de forma intuitiva.  
- Llaves `{}` para mostrar datos dinámicos.
